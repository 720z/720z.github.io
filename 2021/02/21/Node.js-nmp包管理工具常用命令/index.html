Node.js-nmp包管理工具常用命令

国外有一家 IT 公司，叫做 npm, Inc. 这家公司旗下有一个非常著名的网站： https://www.npmjs.com/ ，它是全球最大的包共享平台，你可以从这个网站上搜索到任何你需要的包，只要你有足够的耐心！

到目前位置，全球约 1100 多万的开发人员，通过这个包共享平台，开发并共享了超过 120 多万个包 供我们使用。

npm, Inc. 公司提供了一个地址为 https://registry.npmjs.org/ 的服务器，来对外共享所有的包，我们可以从这个服务器上下载自己所需要的包。

  • 从 https://www.npmjs.com/ 网站上搜索自己所需要的包
  • 从 https://registry.npmjs.org/ 服务器上下载自己需要的包

Node.js-nmp包管理工具常用命令

终端npm命令 作用
npm -v 查看在自己电脑本机的 npm
npm i 包名称 下载一个指定的包
npm view 包名称 查看一个指定包详情信息
npm i 包名称@版本号 下载指定版本的包
npm i 根据 package.json 文件自动下载所有包
npm init -y 快速创建包管理配置文件(package.json)
npm uninstall 包名称 卸载指定的包
npm i –save-dev 包名称@版本号 安装开发测试阶段用到的包
npm i -D 包名称@版本号 同上
npm i -g cnpm 借助cnpm这个包，提供了下载命令 cnpm i 包名称
npm i -g nrm==>nrm ls 安装nrm包，查看所有的可用镜像源
nrm use taobao 将下载包的镜像源切换至taobao镜像
npm i -g i5ting_toc 安装一个能将md文档转成html文件的工具包
i5ting_toc -f ./test.md -o 把test.me文件转为html 并用浏览器打开预览(-o)
npm i -D 包名称 开发测试环境用到的包
npm i -g 包名称 全局包的安装命令
npm uninstall -g 包名称 卸载全局包
npm login 在终端登录npm（需依次输入用户名 密码 游戏）
npm publish 上传发布包至官网
npm unpublish 包名称 –force 强制删除已经发布的包
